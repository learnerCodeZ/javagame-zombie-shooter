**审查抓到的真实 bug（已修）**：
- 🟡 **暂停时计时/难度错乱**：`getElapsedSec()` 原来用**墙钟**(系统时间)，暂停只停 Timer、墙钟照走 → 恢复后存活秒数虚增、刷怪间隔/Brute 门槛被错误跨过。
  - **修复**：改成 `frameCounter / 60`（**逻辑帧**计时），暂停期间 `update()` 不被调用、frameCounter 不增长 → 计时只反映实际游玩时间。

**我额外加的**：`openSettings` 守卫补 `!controller.isRunning()`，防"点设置+同帧死亡"出现设置菜单和死亡结算框叠一起的双弹窗。

**剩余 low（已知小项）**：① `EXIT_W/exitX` 等旧命名残留(纯改名,功能对)；② 连射时 SoundUtil 每次开新音频线可能偶发丢音(已被 try/catch 兜住,不崩)。

## 2. 核心知识点（复盘重点）⭐

### 2.1 数字音频是怎么合成的（本阶段最有价值的知识）
声音 = 一串**采样值**(随时间的气压高低)。代码合成音效就是**自己算这串数字**：

- **采样率** 44100Hz：每秒 44100 个采样点。
- **16bit / mono**：每个采样是一个 -32768~32767 的 short，单声道。
- **波形**：方波 `sign(sin(2πft))`(尖锐,适合"biu")；正弦 `sin(2πft)`(圆润)。
- **频率** f 决定音高(880Hz 高、220Hz 低)；**包络** `exp(-decay·t)` 让声音随时间衰减(不"嘀—"一直响)。
- **频率滑动(hurt)**：用**相位累加** `phase += 2π·f·dt` 而不是直接 `sin(2πft)`，避免相位跳变产生的"咔哒"杂音。

把算出的 short 序列拆成小端序两字节，经 `SourceDataLine` 推给声卡 → 出声。

### 2.2 音量与静音（振幅缩放）
- **音量**：把每个采样的振幅 `× volume/100` → 音量越大振幅越大、声音越响。滑块实时改 `GameSettings.volume`，下个音效即生效。
- **静音**：`muted=true` 时直接 return，不出声。

### 2.3 异常隔离：锦上添花的功能绝不能拖垮核心 ⚠️
音效是非关键功能。`SoundUtil` 整段播放逻辑用 `try/catch(Exception)` 包住 + **守护线程**异步播放：
- 无声卡/线路忙/`LineUnavailableException` → **静默吞掉**，绝不向上抛。
- 异步(守护线程) → **不阻塞** EDT/游戏循环。
> 原则：**可选功能(音效/特效)的失败不能连累主流程**。TestGameLogic 跑 600 帧、shoot 触发 20 次音效，全程不崩，就是靠这个。

### 2.4 游戏暂停 = 停主循环
暂停不是"冻结画面"，而是**停掉驱动 `update()` 的 Timer**：`panel.stop()` → 不再 tick → 游戏状态静止；`panel.start()` → 恢复。配合模态 `JDialog` 菜单，自然实现"开菜单即暂停、关菜单即继续"。

### 2.5 暂停计时的正确性（关键 bug 的教训）⭐
- **墙钟(System.currentTimeMillis)** 会把暂停时间也算进去 → 计时/难度全乱。
- **逻辑帧(frameCounter)** 只在实际游玩时增长 → 暂停不计入 → 正确。
> 凡是"会暂停的游戏"，**用游戏内逻辑时间，别用真实墙钟**。这是个经典坑。

### 2.6 重构：抽公共方法
把"存战绩"从 `settle()` 抽成 `saveCurrentRecord()`，让**死亡结算**和**设置菜单的再来一局/回主菜单**都复用，避免逻辑重复。

## 3. 踩坑 / 注意点

| 坑 | 解法 |
|---|---|
| 暂停时墙钟照走 → 计时/难度错乱 | 改用逻辑帧 `frameCounter/60` |
| 音效失败连累游戏 | try/catch + 守护线程，绝不外泄 |
| 音效阻塞游戏循环 | 异步播放(fire-and-forget) |
| 开菜单+同帧死亡 → 双弹窗 | `openSettings` 守卫加 `!isRunning()` |
| 连射开太多音频线 → 偶发丢音 | (已知；复用一条 line 可优化，本阶段未做) |
| 按钮改"设置"但旧命名 `EXIT_*` 残留 | (cosmetic；功能正确) |

## 4. 怎么用（你来验证）

`run.bat` → 登录 → 开始游戏：
- 右上角点 **"设置"** → 游戏暂停 + 弹菜单

- 射击/命中/受伤/结束时应有**合成音效**(听一下有没有声)
