## 2. 核心知识点（复盘重点）⭐

### 2.1 键盘输入：状态跟踪 vs 单次事件
连续移动要"**按住一直走**"，所以**不能**只在 `keyPressed` 事件里动一下——得记录"哪些键正按住"(用 `Set<Integer> pressedKeys`)，每帧看状态算方向。`keyPressed`→add、`keyReleased`→remove。失焦时 `clear()` 防卡键。

### 2.2 移动数学（方向向量归一化）
```
dx = 右?1:0 - 左?1:0     // 对键抵消(左右同按=0)
dy = 下?1:0 - 上?1:0
若 dx,dy 都非0: len=√(dx²+dy²)=√2 → 各除以 len → 斜走不加速
player.x += dx/len * SPEED;  钳制到画布内
```
**对键抵消**、**斜走归一化**、**边界钳制**三件套，是 2D 移动的标准做法。

### 2.3 寻路：锁定 vs 持续追踪(homing)
- 旧：僵尸出生时锁定玩家位置，之后朝那个**固定点**走 → 玩家一动就全躲开。
- 新：每帧 `zombie.setTarget(玩家当前位置)` 再 `update()` → 僵尸**持续追**当前玩家 → 移动才有躲避价值。

### 2.4 难度曲线（随时间递增，越往后越难）
游戏随存活时间**双维度**自动加压，难度只升不降——玩家不操作，压力也会一直往上走：

**① 刷怪越来越快** — `间隔 = max(30, 90 - 已存活秒数)` 帧（[`GameController.java:168`](../../src/main/java/com/game/game/GameController.java#L168)）
- 开局(t=0)：间隔 90 帧 ≈ 1.5 秒/只；
- 每存活 1 秒间隔减 1 帧，**30 秒**时降到 1.0 秒/只、**60 秒**时压到底线 30 帧 ≈ 0.5 秒/只（前 60 秒刷怪频率涨了 3 倍）；
- 60 秒后封顶在 0.5 秒/只，刷怪频率不再继续加快。

**② 壮汉(Brute)登场** — 满存活 **10 秒**后，每次刷怪有 **18%** 概率换成**更慢、3 血、25 分**的壮硕僵尸，而非普通 1 血绿僵（[`GameController.java:189`](../../src/main/java/com/game/game/GameController.java#L189)）。单只更扛打、更难清理，等效抬升压力。

> 刷怪加速这条线 60 秒后封顶，但壮汉这条线没有上限、且屏幕僵尸越攒越多——所以"越往后越难"的趋势始终成立，只是 60 秒后增速放缓。

一行 `max` 取下限的公式 + 一道概率门槛，就做出"开局轻松、越打越紧张"的曲线。调难度改 `GameController` 头部常量即可：`SPAWN_INTERVAL`(初始间隔) / `MIN_SPAWN`(最快间隔) / `BRUTE_MIN_SEC`(壮汉何时登场) / `BRUTE_CHANCE`(壮汉概率)。

### 2.5 游戏打击感(Juice) 🎯
这是本阶段最大的"超出预期"收获。让游戏有反馈感的常见手段：
- **粒子**：开枪/命中/死亡飞溅 → 让"打中"有视觉确认。
- **震屏**：被撞时画面抖一下 → 强调"受击"。
- **飘字**：`+10` 从僵尸身上飘起 → 得分即时反馈。
- **红屏/闪白**：受伤红屏、命中闪白 → 状态一眼可见。
- **准星/暗角/网格**：提升瞄准精度和画面氛围。

> 这些不改变玩法，但**极大提升手感**——是商业游戏和业余游戏的分水岭之一。

### 2.6 焦点与 EDT
Swing 里"谁接收键盘事件"取决于**焦点**。`requestFocusInWindow()` 要在窗口真正就绪后调，所以用 `SwingUtilities.invokeLater` 推迟一帧，避免"开局 WASD 没反应、点一下才好"。

### 2.7 Graphics2D 进阶
抗锯齿(`RenderingHints`)、半透明(`new Color(r,g,b,alpha)`)、径向渐变(`RadialGradientPaint` 做暗角)、多边形(`fillPolygon` 做耳朵)——纯代码就能画出有质感的画面，不依赖图片。

## 3. 踩坑 / 注意点

| 坑 | 解法 |
|---|---|
| 按一下只动一下 | 用按键**状态集合**，每帧看状态 |
| 斜走比直走快 | 方向向量**归一化**(÷√2) |
| 切窗后角色一直跑 | `focusLost` 清空按键 |
| 开局 WASD 没反应 | `requestFocusInWindow` 用 `invokeLater` 推迟 |
| 僵尸朝老位置走(移动没意义) | 每帧 `setTarget` 到玩家**当前**位置 |
| 刷怪节奏在秒边界轻微抖动 | (已知小瑕疵；改用倒计时计数器更均匀，本阶段未改) |

## 4. 怎么玩（你来验证手感）

`run.bat` → 登录 → 开始游戏：
- **WASD 或方向键**移动玩家(可斜走、对键抵消、不出界)
- **鼠标**瞄准、**点击**射击
- 越打僵尸刷得越快；10 秒后偶尔出**壮汉僵尸**(慢、肉、25 分、头顶血条)
- 打中有粒子+飘字、被撞震屏+红屏、HUD 有血条
- 右上角"退出本局"照常结算
